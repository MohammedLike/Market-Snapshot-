# Market Snapshot

A premium, two-page high-contrast after-market report dashboard for the Indian Stock Market.

## Features
- **Two-Page PDF Export:** Professional institutional-grade layout.
- **Intraday Key Levels:** Staggered SVG visualization of Support/Resistance.
- **Indices Closing:** Tracks Nifty, Bank Nifty, and broader market Advance/Decline.
- **Sentiment Analysis:** High-impact visual markers for market mood.
- **Deep Dive Data:** FII/DII flows, Options OI, and Corporate Results.

## Tech Stack
- React
- Vite
- Framer Motion
- Recharts
- html2canvas & jsPDF (for exports)

## Getting Started
1. `cd market-snapshot`
2. `npm install`
3. `npm run dev`
